# MOIPaperBenchmark

MOIPaperBenchmark is a repository containing scripts and data related to the
paper: _MathOptInterface: a data structure for mathematical optimization
problems_.

## One-time setup

```
$ julia --project -e "import Pkg; Pkg.instantiate()"
$ pipenv install
```
